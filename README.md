# Proyecto_S5
Descripción corta (para el “About” del repo): Análisis estadístico de uso y facturación de 500 clientes de Megaline (2018) para comparar dos planes de prepago (Surf vs. Ultimate). Incluye preparación de datos, métricas mensuales por usuario, visualizaciones y pruebas de hipótesis (t-test) para determinar qué plan genera mayor ingreso promedio.
